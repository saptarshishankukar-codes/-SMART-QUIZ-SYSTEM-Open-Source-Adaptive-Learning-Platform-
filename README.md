# -SMART-QUIZ-SYSTEM-Open-Source-Adaptive-Learning-Platform-
An adaptive learning and assessment platform for schools, colleges, and open communities.
Smart Quiz System is an open-source, Python-based adaptive learning platform designed to provide personalized assessment experiences.
The system dynamically adjusts question difficulty based on user performance, helping learners improve efficiently through intelligent feedback and analytics.
This project promotes Free and Open Source Software (FOSS) by allowing the community to extend question banks, enhance algorithms, and improve features collaboratively.
